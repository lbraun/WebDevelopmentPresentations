What is Node.js

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.
As an asynchronous event driven framework, Node.js is designed to build scalable network applications.

Creating our first Node server

Create a firstNode.js file with the following code:


var http = require('http');

http.createServer(function (req, res) {
    res.writeHead(200, {'Content-Type': 'text/plain'});
    res.end('Hello World\n');
}).listen(3000, "127.0.0.1");

console.log('Server running at http://127.0.0.1:3000/');

Starting up the server

Just type the following in a terminal:


$ node firstNode.js
Server running at http://127.0.0.1:3000/

A message in the screen tells us that the server is running.

Checking if the server is running

Just, open a web browser and type: http://localhost:3000, then press enter.

Stopping the server

Just press Ctrl+C in the terminal.

The server code in detail


var http = require('http'); // The http module is load.

http.createServer(function (req, res) { // We create the server. req is the request object. res is the response object
    res.writeHead(200, {'Content-Type': 'text/plain'}); // This is a header response
    res.end('Hello World\n'); // This is the response content
}).listen(3000, "127.0.0.1"); // It starts to listen

console.log('Server running at http://127.0.0.1:3000/');
